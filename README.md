# Làm Rule Engine Tính giá điện bậc thang

## Đầu vào: Hóa đơn điện (Mã Hợp Đồng, Số điện sử dụng, Khu Vực (1,2,3))

## Yêu cầu: 
 - Khu vực 1: 
   + < 200 số thì 1k
   + 200-500 số thì 1.5k
   + \> 500 số thì 2k
 - Khu vực 2: 
   + < 200 số thì 800đ
   + 200-1000 số thì 1.2k
   + \> 1000 số thì 1.5k
 - Khu Vực 3: 
   + < 1000 số thì 1k
   + \> 1000 số thì 1.3k
